Cars

This website, developed using Next.js, seamlessly integrates two APIs to provide comprehensive information about cars. Utilizing the first API, the site fetches details like name and statistics of various cars. It then taps into a second API to retrieve 3D models of the selected car from different angles. The user-friendly interface enables visitors to search for cars based on model, fuel type, and year, offering a tailored experience. Additionally, the site provides estimated rental costs based on the selected car's specifications, enhancing its practicality for users seeking reliable and detailed information on rental options.

Events

This site showcases an innovative event ticketing platform that provides users with a seamless experience to create and purchase tickets for various events. This platform is built using cutting-edge technologies to ensure efficiency, security, and a user-friendly interface.

Key Technologies:

TypeScript: Utilizing TypeScript for strong typing and improved developer productivity.
Next.js: Employing Next.js for server-side rendering, optimized performance, and a smooth user experience.
Tailwind CSS: Implementing Tailwind CSS for a modern and responsive design, ensuring a visually appealing interface across devices.
Shadcn UI: Integrating Shadcn UI components for enhanced aesthetics and a polished user interface.
Clerk for Authentication: Ensuring secure user authentication through Clerk, providing a seamless and reliable login experience.
MongoDB: Leveraging MongoDB as the database solution for efficient data storage and retrieval, contributing to the platform's scalability.
Stripe for Checkout: Utilizing Stripe for a secure and streamlined checkout process, enabling users to make purchases with confidence.
